# Chadmail template

This is a template for a simple email campaign.

## Getting Started

1. Clone the repository
2. Run `pnpm install`
3. Run `pnpm dev`

## Stack

- Next.js
- Tailwind CSS
- Shadcn UI
- React Hook Form
- Prisma
- PostgreSQL
